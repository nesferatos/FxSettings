FxSettings widget

FxSettings is JavaFX widget for editing settings in your app. It uses POJO as settings containers.

How to start

Just create model of your settings in simple POJO, annotate fields with @Settings and create FXSettings widget with root instance of your model. FXSettings also supports factories for creating lists. More information you can find in my blog(nesferatos.ru)and in FXSettings project webpage (fxsettings.nesferatos.ru)

Maven dependency

<dependency>
    <groupId>ru.nesferatos</groupId>
    <artifactId>fxsettings</artifactId>
    <version>0.1</version>
</dependency>
